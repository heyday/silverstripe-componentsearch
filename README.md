##Installation

###Non-composer

To install just drop the silverstripe-componentsearch directory into your SilverStripe root and run a /dev/build/?flush=1

###Composer

Add the following to your composer.json file:

```json
{
    "require": {
        "heyday/silverstripe-componentsearch": "*"
    }
}
```
